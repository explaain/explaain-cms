How to add Explaain to your site

Include the explaain Javascript

Add this line of code anywhere on your site.

<script src="https://explaain-cms.herokuapp.com/embed/explaain.js"></script>

The code is designed to be cross browser and not to require any dependancies. Everything is under the namespace "explaain" so it should not clash with any Javascript already on your site.

Add one or more elements with the class "explaain"

Example 1

Displaying Headline cards matching the keywords "Donald Trump".

<div class="explaain" data-keywords="Donald Trump"></div>

Example 2

Display a specific card by it's ID.

<div class="explaain"data-id="https://explaain-api-develop.herokuapp.com/Organization/5787d1f6dac9821100609b1f"></div>
